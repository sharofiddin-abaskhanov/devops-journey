# 4-Kun - Docker Compose

## Nima qildik:
Flask va PostgreSQL konteynerlarini
birgalikda ishga tushirdik.

## Fayllar:
- app.py - Flask web ilovasi
- Dockerfile - Image ko'rsatmasi
- docker-compose.yml - Konteynerlar boshqaruvchi
- requirements.txt - Python kutubxonalar

## Buyruqlar:
- docker compose up -d
- docker compose ps
- docker compose down
